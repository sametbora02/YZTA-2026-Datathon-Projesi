# YZTA-2026-Datathon-Projesi

### Yapay Zeka ve Teknoloji Akademisi Datathon Çözümü

Bu depo, Yapay Zeka ve Teknoloji Akademisi (YZTA) kapsamında düzenlenen Datathon yarışması için geliştirilen makine öğrenmesi çözümünü içermektedir.

**Proje İçeriği:**
*   **Veri Seti:** Bilişsel performans ve yaşam tarzı verileri.
*   **Model:** Stacking (XGBoost, CatBoost, LightGBM) Regresyon.
*   **Hedef:** Bilişsel performans skorunu en düşük RMSE ile tahmin etmek.
